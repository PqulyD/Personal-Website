Pneumonia Detection Neural Network

This project uses a convolutional neural network to classify chest X-ray images as either normal or pneumonia. In the notebook, I load the Chest X-Ray Pneumonia dataset from Kaggle, resize every image to 224 by 224 pixels, and normalize pixel values so the model can learn from a consistent input format. I also apply data augmentation to the training set with zoom, shear, and horizontal flips to improve generalization and make the model more robust to variation in the images.

The network itself is a custom CNN built with TensorFlow and Keras. It stacks four convolutional blocks with 32, 64, 128, and 256 filters, using batch normalization, ReLU activations, and max pooling to progressively extract more detailed visual features from each X-ray. After flattening the feature maps, the model passes them through a dense layer with dropout before producing a final sigmoid output for binary classification. The model is trained with the Adam optimizer and binary cross-entropy loss, and the notebook uses a learning-rate scheduler to reduce the learning rate when validation loss stops improving.

For results, the notebook trains on 5,216 training images, validates on 16 images, and evaluates on 624 held-out test images. It tracks both training and validation accuracy and loss across epochs, then measures final test performance on the separate test set. The code also includes a visualization step that shows the CNN's predicted probability for an individual chest X-ray, which makes the model's output easier to interpret. Overall, the project demonstrates a full deep learning workflow for medical image classification, from preprocessing and model design to evaluation and prediction analysis.
