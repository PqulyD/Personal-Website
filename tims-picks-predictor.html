Tims Picks Predictor

Context

On the Tim Horotons App, there is a game that users can play called the Tims NHL Challange. This is a daily game that allows users to pick three NHL players that they think will score thatt day. This not gambling because you are not wagering anything. You have to pick one player from each of three lists. List one contains players who are consistent goal scorers, list two consists of semi-frequent goal scorers, and list 3 has players who do not score that often. If you get one prediction right, you get 25 reward points, two predictions right is 50 rewards points, and getting all three player predictions gets you 100 reward points. Also the app tracks if you have a streak of getting consecutive days of correct predictions. This is important because if you get a correct pick in each day within a span of seven days in a row, you get a coupon on the app that gives you free coffee each day for a week. The app also has a leaderboard that tracks how accurate users are. There are leaderboards that tracks stats of users for the whole season but they also track your monthly stats as well.

What does the program do?

My program takes data from an NHL player stats where it takes each stat and multiplies it my a scalar. Each stat is then summed up for each player. The player with the highest sum is the most likely to score. This is what the formula that find which player is most likely to score:

shots_pg = (form["shots"] / form["games_used"]) if form["games_used"] else 0

goals_pg = (form["goals"] / form["games_used"]) if form["games_used"] else 0

toi = form["avg_toi"]

fatigue_pen = 0.0

if fat["days_rest"] <= 1:

fatigue_pen += 0.25

fatigue_pen += max(0, fat["games_last_6"] - 3) * 0.08

matchup_adj = (opp_ga - 3.2) * 0.25

home_adj = 0.05 if is_home else 0.0

raw = (

1.25 * shots_pg +

0.90 * goals_pg +

0.03 * toi +

matchup_adj +

home_adj

- fatigue_pen

)

When the program is ran, it asks for the user to type in each player within their respected list.

Highlights

  • 76% accuracy (measured in getting at least 1 prediction correct that day)
  • Peaked at 2,756th place on the monthly leaderboard which is the top .54 percentile
  • 1 seven day streak that got me free coffee for a week
