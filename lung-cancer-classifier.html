Gradient Boosted Random Forest Lung Cancer Classifier

This project uses a gradient boosted random forest to classify lung cancer cases based on patient data. In the notebook, I load the lung cancer dataset, preprocess the features, and train the model with various hyperparameters. The notebook also includes a detailed analysis of feature importance and model performance.

The model itself is a gradient boosted random forest built with scikit-learn. It combines multiple decision trees to improve prediction accuracy and reduce overfitting. The notebook uses cross-validation to tune hyperparameters and evaluate the model's performance.

For results, the notebook trains on a dataset of lung cancer cases, validates on a separate set, and evaluates on a held-out test set. It tracks both training and validation accuracy and loss across iterations, then measures final test performance on the separate test set. The code also includes a visualization step that shows the gradient boosted random forest's predicted probability for an individual patient case, which makes the model's output easier to interpret. Overall, the project demonstrates a full machine learning workflow for medical diagnosis, from preprocessing and model design to evaluation and prediction analysis.
